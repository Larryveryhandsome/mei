第一章　緒論

一、計畫緣起

我國過往多處都市計畫公共設施保留地（如公園、綠地、停車場等）長期未能取得或開闢，形成「紙上公設」問題，影響民眾財產權及公共資源配置效率。內政部遂頒布「都市計畫公共設施保留地檢討變更作業原則」，並補助地方政府辦理專案通盤檢討，以檢討不具使用必要性之公設保留地，調整為更適切之分區，以回應實際使用需求、減輕公帑負擔並維護環境。

本案即屬宜蘭縣政府針對「梅花湖風景特定區」辦理之公共設施用地專案通盤檢討，經宜蘭縣都市計畫委員會及內政部都市計畫委員會審議通過。

二、法源依據

  • 《都市計畫法》第26條
  • 《都市計畫定期通盤檢討實施辦法》第14條第4款、第17條第2項但書（情形特殊）
  • 內政部所頒「都市計畫公共設施保留地檢討變更作業原則」

三、審議程序與紀要（摘錄）

  • 宜蘭縣都市計畫委員會審議通過（附件一載有第211次會議紀錄摘錄）

第三章　檢討分析與變更原則

一、都市計畫公共設施保留地檢討變更作業原則（摘要）

依內政部原則與宜蘭縣自訂檢討原則，對各類公設保留地（機關、學校、市場、停車場、體育場、河道、道路等）按是否有具體事業計畫、財務可行性、實際使用需求、服務範圍與替代性進行檢討。不具需求或無法開闢者，得檢討為其他適當分區（如農業區或保護區），或改以整體開發（如市地重劃）取得所需公設。

二、《定檢辦法》第17條第2項但書「情形特殊」之適用

  • 但書所列「情形特殊」包含風景特定區、水源特定區等，其計畫目的具公共開放空間與設施功能，性質與一般市鎮計畫不同
  • 本案屬風景特定區，區內公園、廣場、體育活動空間等公共開放空間之功能，符合但書第二條第一款之審議原則，得不受「通盤檢討後5項公設面積不得低於檢討前」之限制，惟仍應具體提升實質取得與使用效益

三、宜蘭縣公共設施用地專案通盤檢討—本案適用重點

  • 未開闢或不具可行性的公設用地：檢討調整為農業區或保護區（依現況使用、地形坡度、環境敏感度）
  • 以恢復或變更為非都市發展用地（農業區或保護區）為主要處理方式者，應說明維持景觀綠帶與環境保護之具體措施（如退縮維持綠帶）
  • 其餘未取得之公設用地，得併入後續「第三次通盤檢討」一併檢討與推動實質取得

第六章　土地使用分區管制要點

註：本章依計畫書「土地使用分區管制要點」條文整理，僅調整排版與用語。

通則

  1. 本要點依《都市計畫法》第32條及臺灣省施行細則第35條訂定。
  2. 本區劃設之分區與公設用地如下：
    • 第一種遊憩服務設施區、第二種遊憩服務設施區、住宅區、農業區、保護區、第一種宗教專用區、第二種宗教專用區、兒童教養專用區、公園用地、停車場用地、兒童遊樂場用地、綠地、溝渠用地、道路用地、污水處理廠用地。
  3. 綠化與植栽通案要求：
    • 各基地法定空地之綠覆率不得小於50%
    • 每50平方公尺植樹1株（不足5株者以5株計）。退縮部分應綠化。植栽配置應於建築設計階段提出，建造執照審查時檢附，完工後由所有權人或經營人維護，不得任意砍伐或移植。
    • 觀光主管機關得列管綠化狀況並定期稽查，必要時得要求補植。
  4. 公共設施用地之使用：除兒童遊樂場用地外，不適用「都市計畫公共設施用地多目標使用方案」。

各分區管制

（一）第一種遊憩服務設施區

  1. 基地面寬不得小於5公尺；單一基地面積不得小於100平方公尺。
  2. 建蔽率≤40%、容積率≤80%、高度≤3層或10.5公尺。
  3. 基地臨接公共設施用地須退縮10公尺，其退縮部分得計入法定空地。
  4. 停車設置：以實際開發範圍之樓地板面積每100㎡設置1部小客車位，且每10部小客車位增設1部大客車位；小客車位得以3部折算1部大客車位。
  5. 容許用途（舉例）：旅館、零售商店、餐廳、遊客／會議中心、茶藝館、健身房、游泳池、球場等旅遊服務設施。
  6. 屋頂形式：平屋頂覆蓋面積不得逾建築面積二分之一；其餘屋頂部分斜率不得小於25%。

（二）第二種遊憩服務設施區

  1. 建蔽率≤30%、容積率≤60%、高度≤3層或10.5公尺。
  2. 停車比率、退縮與屋頂規範，準用第一種遊憩服務設施區之規定。

（三）住宅區

  1. 建蔽率≤60%、容積率≤120%、高度≤3層或10.5公尺。
  2. 申請建築時，應先完成所屬基地臨接之計畫道路（用地之取得與開闢）。
  3. 屋頂形式：平屋頂覆蓋面積不得逾建築面積二分之一；其餘屋頂部分斜率不得小於25%。

（四）農業區

  1. 以農業生產為主，禁止設置土石方堆置場、廢棄物資源回收貯存場、汽車運輸業停車場（站）、客／貨運站及其附屬設施、加油（氣）站等。
  2. 一般建築（非農舍）：建蔽率≤40%、容積率≤100%。
  3. 農舍：最高3層或10.5公尺；單層最大基底面積≤165㎡；其餘依農業主管法規辦理。
  4. 屋頂形式：平屋頂覆蓋面積不得逾二分之一；其餘屋頂斜率≥25%。
  5. 特別條款（本案新增）：綠地改為農業區之基地，申請農舍時須退縮至原綠地劃設寬度，以維持既有綠帶。

（五）保護區

  1. 以涵養水土、保護特殊地形地勢為主；原則上不得開發。
  2. 自然坡度＞30%之土地不得作為建築使用，並不得計入法定空地。
  3. 建築物高度原則不得超過1層或7公尺。
  4. 申請農舍者：建築物單層最大基底≤165㎡，且不得超過原有（合法）樓地板面積。
  5. 其他經准用為建築使用者：建蔽率≤40%、容積率≤100%；屋頂斜率同上。

（六）第一種宗教專用區

  1. 以宗教建築及其附屬設施為主；不得設置納骨設施。
  2. 建蔽率≤50%、容積率≤100%。
  3. 坡度＞30%之土地不得作建築使用，並不得計入法定空地。
  4. 須設置污水與雨水分流系統；加強水土保持與景觀綠化。

（七）第二種宗教專用區

  1. 以宗教活動及其附屬設施為主；不得設置納骨設施。
  2. 建蔽率≤50%、容積率≤60%、高度≤2層或7公尺。

（八）兒童教養專用區

  1. 以教養、照護、復健等設施為主。
  2. 建蔽率≤50%、容積率≤100%。

（九）兒童遊樂場用地

  1. 以兒童育樂及辦理冬／夏令營等相關活動為主。
  2. 建蔽率≤15%、容積率≤30%；基地面積≥0.5公頃。
  3. 由觀光主管機關擬具管理與使用方案辦理。

（十）污水處理廠用地

  1. 建蔽與容積由縣府主關機關依公共安全與公害防制等公益事項，按實際設置需求擬定。
  2. 臨道路用地退縮5公尺後始得建築。
  3. 場區採開放式參觀動線並以綠化遮蔽，發揮環境教育功能。

（十一）其他公設用地

公園、停車場、綠地、溝渠、道路等依原計畫公設使用管制規定辦理；其細部設計應符合環境、生態與安全之專業基準。
